# Todo Docker App

A full-stack Todo List application built using Django, React, PostgreSQL, and Docker.  
This project demonstrates how to build and run a containerized full-stack application using Docker Compose.

## Features

- Create new todo tasks
- View all tasks
- Update task status
- Delete tasks
- REST API using Django REST Framework
- React frontend for user interaction
- PostgreSQL database
- Docker containerization

## Tech Stack

Backend
- Python
- Django
- Django REST Framework

Frontend
- React.js
- JavaScript
- HTML / CSS

Database
- PostgreSQL

DevOps

- Docker Compose
- Nginx
